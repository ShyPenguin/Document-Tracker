# Document Tracking System
Document Tracking System For Moresco


Jawad Domato
Jean Claude Tancongco
Andrick Aladano

This web-based application will facilitate user management, document creation, tracking, proof of delivery, and reporting functionalities in
the consideration of document tracking system for MORESCO. Anything beyond that is not included.


You can see use the application in this website: https://moresco-1n0y.onrender.com/
For the api of this website: https://moresco-api.onrender.com/ 

To use the application, it requires internet as the database is stored in cloud, the server is deployed as a web-service in Render,
and the front-end is deployed as a web-static in Render. 


Main Libraries Used:


For Front End- React, Redux, Axios


For Back End - Express, Cors, Mongoose, Bcrypt, Nodemon, Dotenv
